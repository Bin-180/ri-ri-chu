# 日日初點餐系統網頁版

這是一個可直接開啟的靜態網頁點餐系統，不需要安裝套件或啟動伺服器。

## 開啟方式

直接用瀏覽器開啟 `index.html`。

## 功能

- 菜單分類切換
- 點選加入購物車
- 數量加減與清空訂單
- 內用 / 外帶切換
- 內用自動計算 10% 服務費
- 桌號 / 姓名與訂單備註
- 現金、Line Pay、刷卡付款選項
- 送出訂單後顯示訂單編號與金額
- 訂單資料會輸出在瀏覽器 Console，方便後續串接後台或印單機

## 修改菜單

菜單資料在 `app.js` 的 `menuItems` 陣列中，可直接調整品項名稱、分類、描述、價格與色彩。
